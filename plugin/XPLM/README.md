# X-Plane SDK

Drop the unpacked X-Plane SDK here, so that:

```
plugin/XPLM/CHeaders/XPLM/XPLMDefs.h
plugin/XPLM/Libraries/Win/XPLM_64.lib    (Windows builds)
plugin/XPLM/Libraries/Mac/XPLM.framework (macOS builds)
```

all resolve. SDK download: https://developer.x-plane.com/sdk/plugin-sdk-downloads/

The SDK is not redistributed in this repository.
